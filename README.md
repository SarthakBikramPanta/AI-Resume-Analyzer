# AI Resume Ranker 🚀

An entry-level NLP project designed to automate the initial screening process of job applications using vector space modeling.

## 🛠️ How it Works
This project uses **Term Frequency (CountVectorizer)** to convert text into numerical vectors. It then calculates the **Cosine Similarity** between the Job Description vector and the Resume vector to determine a match percentage.

## 🧠 Key AI Concepts Used
* **Natural Language Processing (NLP):** Processing raw text for machine analysis.
* **Vectorization:** Transforming words into a mathematical "Bag of Words" representation.
* **Cosine Similarity:** Measuring the distance between two vectors in multi-dimensional space to determine similarity.
